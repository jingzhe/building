# Buildings backend

## How to run the service
Use IntelliJ IDEA to open the project and run the service, or use command line bellow
### Compile
```
mvn compile
```
### Start the service
```
mvn spring-boot:run
```
### Run the test cases
```
mvn test
```

### OpenApi documentation
http://localhost:8081/webjars/swagger-ui/index.html
